SmartRoute: Shortest Route Finder

How to Use This Tool

1. Add Nodes: Click anywhere on the map to place a new node (intersection).
2. Connect Nodes: Right-click on a starting node, then right-click on an ending node to create a path.
3. Find Route: Use the controls below to select your 'From' and 'To' nodes, then click "Start Route".
